# opti.node AI Coding Instructions

Django + React monorepo for the opti.node energy management platform.

## Critical Rules

- Always check current directory with `pwd` before running commands
- Compare branches against `develop`, not `main`
- Check existing patterns in similar components/modules before implementing new features
- Run `npm`/`npx` commands from within `frontend/` — NOT from the project root

## Communication style

- Be direct and honest — flag problems even if they're uncomfortable
- Push back when something seems wrong; don't silently agree with mistakes
- Flag unclear or risky points proactively, before they become problems
